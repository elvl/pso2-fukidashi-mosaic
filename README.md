#はじめに
この Avisynth スクリプトは PSO2 の動画の吹き出しにモザイクをかけます。

このスクリプトの動作には、Avisynth 2.6.0 以上と masktools 2.0 が必要です。

マニュアルは [Wiki](https://github.com/elvl/pso2-fukidashi-mosaic/wiki) にあります。
